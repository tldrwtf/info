# Learning Progression Guide

This guide outlines a recommended learning path through the Full Stack Learning Hub.

## Quick Reference Card

| Stage | Focus | Key Resources |
|-------|-------|---------------|
| 1. Foundation | HTML, CSS basics | HTML Cheat Sheet, CSS Cheat Sheet |
| 2. Python Basics | Syntax, control flow | Python Basics Cheat Sheet |
| 3. Data Structures | Lists, dicts, algorithms | Data Structures Cheat Sheet |
| 4. OOP | Classes, inheritance | OOP Cheat Sheet |
| 5. Databases | SQL, SQLAlchemy | SQL and SQLAlchemy Cheat Sheet |
| 6. Web Backend | Flask, APIs | Flask REST API Development Guide |
| 7. JavaScript | DOM, async | JavaScript Basics Cheat Sheet |
| 8. React | Components, state | React Basics Guide |

---

## Stage 1: Web Foundations

Start with HTML structure and CSS styling:

1. [HTML Cheat Sheet](../cheatsheets/HTML_Cheat_Sheet.md) - Tags, structure, attributes
2. [CSS Cheat Sheet](../cheatsheets/CSS_Cheat_Sheet.md) - Selectors, box model
3. [CSS Layout Guide](CSS_Layout_Guide.md) - Flexbox and Grid basics
4. [Bootstrap Cheat Sheet](../cheatsheets/Bootstrap_Cheat_Sheet.md) - Grid system, components

**Learning Goals:**
- Understand HTML document structure and semantic elements
- Apply CSS selectors and the box model
- Create responsive layouts with Flexbox and Grid
- Use Bootstrap for rapid prototyping

---

## Stage 2: Python Fundamentals

Build programming foundations:

1. [Python Basics Cheat Sheet](../cheatsheets/Python_Basics_Cheat_Sheet.md) - Variables, loops, functions
2. [Functional Programming Cheat Sheet](../cheatsheets/Functional_Programming_Cheat_Sheet.md) - Map, filter, lambdas
3. [Error Handling Cheat Sheet](../cheatsheets/Error_Handling_Cheat_Sheet.md) - Exceptions, try/except

**Learning Goals:**
- Write Python functions with proper parameters and return values
- Use control flow statements (if/elif/else, for, while)
- Apply functional programming concepts (map, filter, list comprehensions)
- Handle exceptions gracefully

---

## Stage 3: Data Structures and Algorithms

Master data organization and efficiency:

1. [Data Structures Cheat Sheet](../cheatsheets/Data_Structures_Cheat_Sheet.md) - Lists, dicts, sets
2. [Big O Notation Cheat Sheet](../cheatsheets/Big_O_Notation_Cheat_Sheet.md) - Complexity analysis
3. [Algorithms Guide](Algorithms_Guide.md) - Sorting, searching
4. [Linked Lists and Custom Data Structures Guide](Linked_Lists_and_Custom_Data_Structures_Guide.md) - Custom data structures

**Learning Goals:**
- Choose appropriate data structures for different use cases
- Analyze time and space complexity with Big O notation
- Implement common algorithms (sorting, searching)
- Build custom data structures (linked lists, stacks, queues)

---

## Stage 4: Object-Oriented Programming

Apply OOP principles:

1. [OOP Cheat Sheet](../cheatsheets/OOP_Cheat_Sheet.md) - Classes, inheritance, polymorphism
2. [Advanced Python Cheat Sheet](../cheatsheets/Advanced_Python_Cheat_Sheet.md) - Metaclasses, descriptors, decorators

**Learning Goals:**
- Design classes with proper encapsulation
- Use inheritance and composition appropriately
- Apply polymorphism and abstraction
- Understand advanced patterns (decorators, context managers)

---

## Stage 5: Databases and ORM

Persist and query data:

1. [SQL DDL Guide](SQL_DDL_Guide.md) - Schema definition
2. [SQL Advanced Queries Guide](SQL_Advanced_Queries_Guide.md) - Complex queries
3. [SQLAlchemy CRUD Guide](SQLAlchemy_CRUD_Guide.md) - ORM basics
4. [SQLAlchemy Relationships Guide](SQLAlchemy_Relationships_Guide.md) - One-to-many, many-to-many
5. [SQLAlchemy Advanced Patterns Guide](SQLAlchemy_Advanced_Patterns_Guide.md) - Production patterns

**Learning Goals:**
- Design database schemas with proper relationships
- Write efficient SQL queries with joins and aggregations
- Use SQLAlchemy ORM for Python-database interaction
- Understand relationship patterns and session management

---

## Stage 6: Backend Development

Build APIs with Flask:

1. [Flask REST API Development Guide](Flask_REST_API_Development_Guide.md) - RESTful design
2. [API Authentication Guide](API_Authentication_Guide.md) - JWT, Bearer tokens
3. [Flask Advanced Features Guide](Flask_Advanced_Features_Guide.md) - Caching, rate limiting
4. [Python API Testing Guide](Python_API_Testing_Guide.md) - Testing patterns
5. [Flask API Build Secure Ship Handbook](Flask_API_Build_Secure_Ship_Handbook.md) - Production deployment

**Learning Goals:**
- Design RESTful APIs following best practices
- Implement authentication with JWT and Bearer tokens
- Add caching and rate limiting for production
- Write comprehensive API tests
- Deploy Flask applications securely

---

## Stage 7: JavaScript and DOM

Add interactivity:

1. [JavaScript Basics Cheat Sheet](../cheatsheets/JavaScript_Basics_Cheat_Sheet.md) - Variables, types, loops
2. [JavaScript Functions Guide](JavaScript_Functions_Guide.md) - Functions, callbacks
3. [DOM Manipulation Guide](DOM_Manipulation_Guide.md) - Element manipulation
4. [JavaScript Fetch API Guide](JavaScript_Fetch_API_Guide.md) - HTTP requests
5. [JavaScript LocalStorage Guide](JavaScript_LocalStorage_Guide.md) - Browser storage

**Learning Goals:**
- Write JavaScript with modern ES6+ syntax
- Manipulate the DOM for dynamic UIs
- Make HTTP requests with the Fetch API
- Persist data with LocalStorage
- Handle asynchronous operations with async/await

---

## Stage 8: React and Modern Frontend

Build component-based UIs:

1. [React Basics Guide](React_Basics_Guide.md) - Components, props, state
2. [React Context State Management Guide](React_Context_State_Management_Guide.md) - Context API
3. [React Router Navigation Guide](React_Router_Navigation_Guide.md) - Routing
4. [React to Fullstack Track](React_to_Fullstack_Track.md) - React 19, Next.js 16

**Learning Goals:**
- Build reusable React components
- Manage state with hooks (useState, useEffect, useContext)
- Implement client-side routing
- Lift state up and use controlled components
- Understand modern React patterns and best practices

---

## Stage 9: DevOps and Production

Deploy and maintain:

1. [Git Version Control Guide](Git_Version_Control_Guide.md) - Version control
2. [CI/CD Pipeline Guide](CI_CD_Pipeline_Guide.md) - Automation
3. [Docker and Containerization Guide](Docker_and_Containerization_Guide.md) - Containers

**Learning Goals:**
- Use Git for version control and collaboration
- Set up CI/CD pipelines for automated testing and deployment
- Containerize applications with Docker
- Understand production deployment best practices

---

## Tips for Success

1. **Follow the order** - Each stage builds on previous knowledge
2. **Practice actively** - Work through the code examples, do not just read
3. **Build projects** - Apply concepts in real applications
4. **Use the cheatsheets** - Reference them while coding
5. **Read the glossary** - Clarify unfamiliar terms in the [Glossary](../GLOSSARY.md)
